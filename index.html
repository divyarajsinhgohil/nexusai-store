🚀 How to Go Live
DEPLOY YOUR NEXUSAI STORE IN MINUTES
Deploy Your NexusAI Store Live
1
Save This File
Right-click this page → Save As → choose index.html. This single file contains your entire working website — no server or database required for basic hosting.
2
Choose a Free Hosting Platform
All of these offer 100% free static hosting. Just upload your index.html file:
🔷 Netlify
FREE forever
Drag & drop your file at app.netlify.com/drop. Live in 30 seconds. Custom domain free.
⬛ Vercel
FREE forever
Upload via vercel.com. Best performance. Auto HTTPS. Global CDN.
📄 GitHub Pages
FREE forever
Push to GitHub repo, enable Pages. URL: username.github.io/nexusai
🔥 Firebase Hosting
FREE tier
Google's CDN. Fast. Run firebase deploy after init.
3
Netlify Drop (Easiest — 30 Seconds)
No account needed to start:
1. Go to: https://app.netlify.com/drop 2. Drag your index.html file onto the page 3. Netlify gives you a live URL instantly! Example: https://random-name-12345.netlify.app 4. (Optional) Sign up free to keep the site + custom URL
4
GitHub Pages (For Permanent Hosting)
1. Create account at github.com (free) 2. Click "New Repository" → name it: nexusai-store 3. Upload your index.html file 4. Go to Settings → Pages → Source: main branch 5. Your site is live at: https://YOUR-USERNAME.github.io/nexusai-store
5
Custom Domain (Optional)
Buy a domain from Namecheap (~$10/year) or Google Domains, then connect it to your Netlify or GitHub Pages site for free. Example: nexusai.yourname.com
6
Important: Change Admin Password!
Before going live, change the admin credentials in the HTML file. Find this line in the <script> section and update it:
// Find in the JS code: const ADMIN = { email: 'admin@nexusai.com', password: 'admin123' }; // Change to your own secure credentials: const ADMIN = { email: 'youremail@gmail.com', password: 'YourSecurePass123!' };
7
For Real Database (Advanced)
This site currently uses browser localStorage. For a production app with server-side data shared across ALL users, connect to Firebase Firestore (free tier) or Supabase (free tier). Both have free JavaScript SDKs you can drop into this file.
// Upgrade path: // 1. Firebase Firestore (Google) - console.firebase.google.com // 2. Supabase (Postgres) - supabase.com // 3. PlanetScale (MySQL) - planetscale.com // All have generous free tiers!
